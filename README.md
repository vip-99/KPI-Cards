# KPI-Cards


###KPI Cards Explanation

**Total Sales ($122,509)**
Displays the total revenue for the selected filters (Category, Channel, and Month).

**MoM % Change (-19.94%)**
Shows the Month-over-Month performance. Sales decreased by 19.94% compared to the previous month.

**YoY % Change (+56.90%)**
Shows the Year-over-Year growth. Sales increased by 56.90% compared to the same month last year.

**Quantity (501)**
Total units sold, with regional contribution shown in the donut chart (North, South, West, East).



**These KPIs help quickly track revenue performance, growth trends, and regional distribution.**
